# Pruebas unitarias (tests)

Este directorio contiene las pruebas unitarias de la biblioteca.

Los archivos incluidos definen distintos casos de prueba que permiten verificar el comportamiento de los algoritmos y estructuras de datos implementados en los módulos de la biblioteca.

Las pruebas deben pensarse para ejecutarse automáticamente y facilitar la validación del código ante cambios o nuevas implementaciones.

Este directorio contiene dos subdirectorios para contener las pruebas de los algoritmos y las estructuras de datos por separado.

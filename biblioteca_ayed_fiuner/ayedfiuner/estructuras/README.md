# Código fuente de estructuras (source)

Este directorio contiene las implementaciones de estructuras de datos de la biblioteca.

Cada módulo define uno o más Tipos Abstractos de Datos (TAD), representados mediante clases que encapsulan su estado y las operaciones disponibles para interactuar con ellos.

Estas estructuras deben diseñarse para ser reutilizadas desde otros módulos de la biblioteca o desde proyectos externos.

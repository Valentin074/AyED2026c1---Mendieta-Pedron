# Código fuente (source)

En este directorio se encuentran los archivos de código fuente que implementan los algoritmos y las estructuras de datos de la biblioteca.

Aquí se definen las clases (con sus atributos y métodos), funciones y otros elementos auxiliares necesarios para el funcionamiento de la biblioteca. 

El código contenido en este directorio está pensado para ser reutilizado e importado desde otros proyectos.

El contenido de este directorio se organiza en dos subdirectorios: "algoritmos" y "estructuras"

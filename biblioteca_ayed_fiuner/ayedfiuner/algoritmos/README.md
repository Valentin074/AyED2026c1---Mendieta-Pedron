# Código fuente de algoritmos (source)

Este directorio contiene las implementaciones de algoritmos de la biblioteca.

Los módulos aquí incluidos definen funciones y, en algunos casos, clases auxiliares que encapsulan distintos algoritmos utilizados en el estudio de Algoritmos y Estructuras de Datos.

Estos algoritmos deben diseñarse para ser reutilizados e importados desde otros módulos de la biblioteca o desde proyectos externos.

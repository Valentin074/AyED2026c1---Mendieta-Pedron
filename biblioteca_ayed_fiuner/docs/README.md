# 📝Documentación de la biblioteca

Este directorio contiene documentación complementaria de la biblioteca.

Aquí se deben incluir documentos que describen los criterios de diseño adoptados, las convenciones de implementación y otros aspectos conceptuales relevantes para comprender la organización y el uso de la biblioteca.

Entre otros temas, la documentación puede incluir:

- criterios de diseño de las estructuras de datos
- convenciones de nombres y organización de módulos
- decisiones de implementación generales
